# Swadesh Analyzer

A simple web app to compare languages using the Swadesh list.

## Features
- Compare two languages
- Calculates similarity %
- Highlights matches and near matches
- Uses Levenshtein distance

## How to Run
1. Download repo
2. Open index.html
